Projet C-WildWater
Ce projet permet de réaliser la synthèse de données d’un système de distribution d’eau potable à partir d'un fichier CSV volumineux. Il combine un script Shell pour la coordination des differentes étapes et un programme en C optimisé pour le traitement de millions de lignes.
